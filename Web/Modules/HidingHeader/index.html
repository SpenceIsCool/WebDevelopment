So heres, the deal. in the "body" tag of your html script there is the option for a "header" tag. This is used by search engines when people are attempting to find your website. Accordingly, you should always have a header tag and you should always have an h1 tag inside of it. This Text in the h1 tag should be the name of your website.


Many find the look of this to be "unattractive". So these following sites show a simple way to remove this header from view.
